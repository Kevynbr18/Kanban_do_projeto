Sistema de Reservas
Este documento lista e descreve todas as páginas incluídas na entrega final do sistema, separadas por perfil Professor e Aluno, além das páginas comuns de Login, Cadastro e Esqueci a Senha.

Páginas Comuns
Estas páginas são acessíveis tanto para professores quanto para alunos.

1. Tela de Login
Tela inicial para autenticação de usuários. Permite acesso ao ambiente do Professor ou do Aluno conforme credenciais.

2. Tela de Cadastro
Formulário para criação de nova conta no sistema.

3. Tela de Esqueceu a Senha
Página para recuperação de senha via e-mail.

Professor — Páginas Pertencentes ao Professor
1. Tela de Login do Professor
Página específica para autenticação do professor.

2. Calendário do Professor
Exibe reservas realizadas pelo professor, disponibilidade e visão geral mensal/semanal.

3. Salas Disponíveis
Lista todas as salas atualmente livres para reserva, com filtros e visualização rápida.

4. Salas Pendentes
Exibe solicitações de reserva que ainda estão aguardando confirmação.

5. Histórico de Agendamentos
Lista completa de eventos e reservas já concluídos ou expirados.

6. Salas Ocupadas
Mostra as salas que já estão ocupadas no momento, com horários e responsáveis.

7. Feedback do Professor
Página onde o professor pode enviar opiniões, avaliações ou relatar problemas ao sistema.

8. Minhas Reservas
Área pessoal com todas as reservas realizadas pelo professor, incluindo status, horários e opções de gerenciamento.

Aluno — Páginas pertencentes ao aluno
1. Tela de Login do Aluno
Página dedicada à autenticação dos alunos.

2. Fazer Nova Reserva
Página onde o aluno escolhe sala, data e horário para registrar uma nova reserva.

3. Feedback do Aluno
Área onde o aluno pode enviar sugestões, avaliações ou relatar problemas.

4. Minhas Reservas
Página que lista todas as reservas do aluno, com opções de cancelamento ou visualização de detalhes.

5. Calendário do Aluno
Exibe de forma visual todas as reservas feitas pelo aluno e a disponibilidade das salas.
